# 02 SQL DEEP DIVE
Tools: 
- DBMS PostGre
- RDBMS Valentina DB

## READING DATA
- pattern :
```sql
SELECT <COLUMN> FROM <TABLE>
```

## RENAME COLUMNS
- pattern :
```sql
SELECT <COLUMN> as <NEW_COLUMN_NAME> FROM <TABLE>
```

## CONCATENATION
- pattern :
```sql
SELECT CONCAT(<COLUMN>, <COLUMN>) as <NEW_COLUMN_NAME> FROM <TABLE>
```
